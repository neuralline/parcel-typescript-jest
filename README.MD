# Parcel-TypeScript testing setup

Setting up Typescript testing could be daunting sometimes so I've made this simple setup template

## Installation

```sh
git clone https://github.com/neuralline/parcel-typescript-jest.git
npm install
# or
yarn install
```

```sh
typescript build:
 > yarn tsc
parcel live server:
 > yarn start
build:
 > yarn build

```
